# learning-journey

A long-running notes dump of things I discover while learning — engineering, career, books, odd bits of history. The bar is short: too small for a blog post, too useful to forget.

Files are dated and grouped by theme (`stories/`, `books/`, `linux/`, `daily-archive/`). Read in any order; nothing here is polished and most of it is dated. Pull requests for new `today-i-learned` entries are welcome.

GPL-3.0.
